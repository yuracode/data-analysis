# 最終成果物

## ビジネス背景
今年度より継続率を改善したい。ただし、追加で投入できる予算と人員は限られている。
どのユーザー層に、どんな支援を、どのタイミングで行うべきかを、
過去3四半期分のユーザーデータから根拠を示して提案してほしい

## 確定した問
### 主問い
 - 入会初期(一ヶ月～三か月)のどの行動指標とどの機能利用が六か月後の会員継続に最も強く影響するか？

### サブ問い
- 一ヶ月から三ヶ月の運動頻度低下は継続率にどのくらい影響あるのか？
- アプリのログイン率やプッシュ通知on率が高いと継続率は高いのか？
- トレーナー相談は継続率に貢献しているのか？

## 分析ストーリー
### 結論(仮)
- 半年後の継続を左右するのは、リアルな努力（運動頻度の維持）や高負荷なサポート（トレーナー相談）ではなく、「入会初期におけるアプリの習慣化と、超低負荷なデジタル・エンゲージメントの構築」であるという仮説を検証する。
一般的に「トレーナー相談が継続を支えている」見られるが、これは継続意欲の高い層が利用しているだけの「選択バイアス」である可能性を疑っている。モチベーションが低下する前提で、いかに初期のログイン率（85%以上）を担保し、プッシュ通知やバッジで受動的な接点を維持できるかが本質的な離脱防止ラインになると予想し、データからその相関・分布を裏付けます。

### 根拠
- 1ヶ月目のログイン率が低い（モチベーションが下がっている）ユーザー群においては、トレーナー相談の有無によって6ヶ月後の継続率（continued）に「有意な差が出ない」ことを確認します。逆に、相談を利用しているのは初期ログイン率が極めて高い層に集中していることを示し、トレーナー相談は継続の「原因」ではなく、高モチベーションの「結果」に過ぎないことを裏付けます。

- 1ヶ月目と3ヶ月目の「週の運動回数」を引き算し、ほとんどの会員がマイナス（回数が減っている）のエリアに集中しているグラフを示します。これにより、リアルな運動回数をキープさせること自体の難易度の高さをあらかじめデータで正しく認識し、「ジムに来させること（リアルな努力）」だけに頼る施策の危険性がある。

-運動頻度が下がってしまったサボり気味のユーザーのなかでも、「プッシュ通知ON」や「バッジ獲得数が1〜2個以上」の層は、通知OFF・バッジ0個の層に比べて、6ヶ月後の継続率が目に見えて高く踏みとどまっている傾向をデータで炙り出します。「ジムに行かない日でも、アプリをなんとなく開いちゃう、通知には反応する」というスマホの中でのゆるい繋がりこそが、最終的な解約を防ぐもう一つの強力な命綱になる。

### 反論
- プッシュ通知をONにしている人の継続率が高いのも、トレーナー相談と同じで『もともとやる気がある熱心な会員だから、最初の設定で通知をONにしただけ（選択バイアス）』じゃないの？通知自体に引き止め効果があるとは言えないのでは？
- =>もし、やる気が下がりかけている中・低モチベーション層において、プッシュ通知ONのユーザーのほうが目に見えて踏みとどまっている（継続している）データが確認できれば、「プッシュ通知は単なるやる気の現れではなく、離脱を引き止める『アプリからの介入効果（命綱）』として機能している」と証明できる

- 3ヶ月目に運動頻度が減っている（workout_changeがマイナス）からといって、みんながサボっていると決めつけるのは違うのでは？アプリ内の自宅トレーニング動画や食事管理が優秀だから、ジムに行く回数を減らして家で効率よく続けられている（ポジティブな減少）可能性もあるはずだ。
- =>もし運動回数が減っても継続している人が、3ヶ月目も高いログイン率を維持していれば、「ジムには行かなくなったが、アプリを通じたエンゲージメント（自宅利用や記録）は維持できている＝ポジティブな減少である」と言えます。逆にログイン率も一緒に死んでいれば、それはただの「サボり（離脱の予兆）」だと判定できる


## 経営層に出すグラフ・表のモック
- 「トレーナー相談」の選択バイアス検証マトリクス
-  運動減少層における「アプリ（通知・バッジ）」の引き止め効果(棒グラフ)

## 判断基準
アプリ内CRM（通知強制ON・初期バッジ獲得）の強化施策
- go 運動頻度が減ったユーザーの中で、「プッシュ通知ONかつバッジ1個以上」のグループ（B群）の継続率が、「通知OFFかつバッジ0個」のグループ（A群）に対して【プラス15ポイント以上】高い場合。
- nogo 両グループの継続率の差が10ポイント未満、あるいはどちらも同じように継続率が低い場合。

対人サポート（トレーナー相談）の運用見直し
- go 1ヶ月目のログイン率が「低（サボり気味）」のユーザーにおいて、トレーナー相談が「あり」の人の継続率が、「なし」の人よりも【プラス15ポイント以上】高い場合。
- nogo ログイン率が「低」のユーザーにおける、相談「あり・なし」の継続率の差が5ポイント未満（ほぼ同じ）だった場合。

## 結論の汎化可能性
 - この分析から得られる「運動を頑張らせるより、アプリの初期ログイン率や通知が命綱になる」という仮説が正しいと証明されたとしても、その結論をそのまま他のジャンルのアプリ（学習アプリや、食事管理専門アプリなど）や、異なるターゲット層（例：スマホ操作に慣れていないシニア層向けの対面フィットネスなど）にそのまま当てはめる（汎化する）ことはできません。
あくまで「アプリをハブとして展開する、この4プランのフィットネスサービス」における特有のユーザー心理・ドメイン知識として結論を扱う必要があります。この不確実性を超えて施策の確度を100%にするためには、本分析の結果をもとに、特定のユーザーグループを対象とした実際の「A/Bテスト（介入実験）」へとステップを進める必要があります。
