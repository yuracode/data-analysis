### セクション1：マーケティングファネル

**マーケティングファネル（購買漏斗）**：顧客が認知から購買・リピートに至るプロセス

```
認知（Awareness）
    ↓
興味・関心（Interest）
    ↓
検討（Consideration）
    ↓
購買（Purchase）
    ↓
リピート・推奨（Loyalty）
```

各段階でのデータ指標例：

| ステージ | 指標例 |
|---------|--------|
| 認知 | インプレッション数・リーチ数 |
| 興味 | クリック率（CTR）・滞在時間 |
| 検討 | カート追加率・比較ページ閲覧数 |
| 購買 | コンバージョン率（CVR）・客単価 |
| リピート | リテンション率・LTV |

### セクション2：RFM分析

**RFM分析**：顧客をRecency・Frequency・Monetaryの3軸でセグメントする手法

| 軸 | 意味 | 指標例 |
|----|------|--------|
| Recency | 最終購買からの経過日数 | 直近購買日 |
| Frequency | 購買頻度 | 購買回数 |
| Monetary | 購買金額 | 累計購買額 |

```python
import pandas as pd
import numpy as np

# サンプルデータ（実際はトランザクションデータを使用）
# orders: customer_id, order_date, amount

snapshot_date = pd.Timestamp('2025-01-01')

rfm = orders.groupby('customer_id').agg(
    Recency   = ('order_date', lambda x: (snapshot_date - x.max()).days),
    Frequency = ('order_date', 'count'),
    Monetary  = ('amount', 'sum')
).reset_index()

# スコアリング（5段階）
for col in ['Recency', 'Frequency', 'Monetary']:
    ascending = (col == 'Recency')  # Recencyは小さいほど良い
    rfm[f'{col}_score'] = pd.qcut(rfm[col], q=5, labels=[5,4,3,2,1] if ascending else [1,2,3,4,5])
```
