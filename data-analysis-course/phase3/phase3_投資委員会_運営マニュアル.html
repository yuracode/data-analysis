【学生向け】投資委員会 運営マニュアル（自走版）

これは 君たち（学生）が見るマニュアルです。先生が読み上げる台本ではありません。
先生は最初のガイダンスだけを行います。そのあとの90分は、この1枚を見ながら君たちだけで回します。
役割ごとの詳しい動きは、それぞれの役割マニュアルへ（→ 後半「7. 役割マニュアルへ」）。
ルールの大もと：[[phase3_投資委員会_進行台本]]

ここまでが先生／ここから先は君たちだけ

┌─ 先生がやること（最初の数分だけ） ─────────────┐
│ ・今日のゴールと「投資委員会」の趣旨を説明        │
│ ・このマニュアルと役割マニュアルの場所を案内      │
│ ・「では、あとは自分たちで」と宣言して退場/見守り │
└──────────────────────────────┘
            ↓ ここから下は、君たちだけで進める
┌─ 君たちがやること（90分・自走） ──────────────┐
│ ・進行役を1人決める                          │
│ ・このマニュアルの「4. 全体タイムライン」を回す  │

大事な前提：この授業に審判（先生）はいません。「良い／悪い」は先生ではなく、
委員の多数決と観察できた事実が決めます。だから君たちだけで完結できます。

1. 30秒で全体像（これだけ掴めば動ける）

  • 各自は FitLoop社の「事業部長」。月40万円の予算を、仲間の前で奪い合います。
  • 自分の分析を 1枚で提案発表 → 仲間（投資委員）が 投資カードで投資先を選ぶ。
  • 60万円（投資カード12枚）以上集めた案だけが「資金がついた＝Go」。届かない案は見送り。
  • 委員は黙って買うだけでなく、鋭い反論で穴を突くと得点（レッドチーム点）。
  • 狙うのは順位だけではない。自分が宣言した段（L1/L2/L3）に届いたかが本当の評価軸。

ひとことで言うと「経営者にGoをもらう」を「仲間の委員会で予算を勝ち取る」に置き換えた回です。

2. 今日狙う段とは（L1 / L2 / L3）— 自己採点の前に読む

段（だん）＝「自分の分析を、価値の鎖のどこまで自力で登れたか」。手助けの量ではありません。
救済ではなく梯子。同じ1本の提案が L1→L2→L3 と育っていきます。下の例で見れば一瞬で分かります。

同じ1本の提案が、こう育っていく（FitLoopの例）

段 ひとことで FitLoopでの具体例（同じ提案が育っていく） ルーブリック目安
L1（床） 問いを1本立て、グラフ1枚で語れる 「プッシュ通知ON層は、OFF層より継続率が高い」をグラフ1枚で示せた。質問が1つ来ても答えられる ≈30 / 60
L2（標準） 数字でGo/No-Goを言い、バイアスを自覚 「ON層62% vs OFF層48%＝14pt差（n=○○）。継続率55%を超えればGo」。さらに「push_opt_in は自分でONにした人＝自己選択。だから因果とは言えない」と、聞かれる前に自分で添える ≈45 / 60
L3（天井） 反論に耐え、データに従って結論を組み替える 「それ自己選択でしょ？」と突かれ、低モチベ層だけで切ったら差が消えた。→ "全員にプッシュ"案を捨て、"高モチベ層に絞る"案にその場で組み替えて再提示 ≈55以上 / 60

ポイント：L1→L2 は「数字とバイアス但し書きを足す」、L2→L3 は「反論を浴びて結論を変える勇気」。
段が上がるほど、足すのは“分析量”ではなく“正直さと打たれ強さ”です。

「ここまで出来たら次の段」自己チェック

  • L1に届いたサイン：「継続率を上げたい」ではなく「◯◯と△△に関係はあるか？」の形で問いを言える／グラフが自分の問いに対応している（飾りのグラフではない）。
    • まだ Go/No-Go の数値がない・バイアスに触れていない → 次は L2。
  • L2に届いたサイン：「この数字が出ればGo／出なければNo-Go」の閾値を実データの数字で言える／聞かれる前に自己選択を自分で但し書きできる。
    • 痛い反論が来ると黙る・結論に固執してしまう → ここを越えれば L3。
  • L3に届いたサイン：バイアス攻撃を受けても「◯◯で切ったら差が消えた。だから提案をこう変える」とその場で組み替えられる／委員としても他人の穴（サンプル数・閾値・自己選択）を1〜2件突ける。

使い方（3つだけ）

  1. 段は固定ではありません。コマ15ではL1、最終回ではL3、と毎回ひとつ上を狙ってよい。
  2. 走る前に「今日はL2を狙う」と宣言してから自己採点（→ 次の「3. 始める前の3分セットアップ」）。
  3. L1で全部そろわなくても減点ではなく「次の一手」。床は誰でも立てる場所です。

3. 始める前の3分セットアップ（自走の起動儀式）

先生が抜けたら、まずこの3つを済ませてから時計を回し始めます。

① 進行役を1人決める

  • その回、提案発表しない人が1人いれば、その人が進行役（中立だから）。
  • いなければ じゃんけんで1人決める。専門知識は要りません。タイマーを押すだけの係です。
  • 進行役以外は全員「発表者」かつ「投資委員」を兼ねます（自分の番以外は委員）。

② 配る・開く

  • 投資カードを各委員に 8枚ずつ配る（物理チップ、または配分フォームのリンクを共有）
  • リーダーボード（スプレッドシート）を開いて全員に見える形に
  • 自己採点シートを全員に配る

③ 各自が「今日狙う段」を宣言（3分）

  • 自己採点シートの先頭に L1 / L2 / L3 のどれを狙うかを書く
  • 提案発表前の自己採点（/60）もここで記入

ここまで終われば準備完了。あとはタイムラインを上から流すだけです。

4. 全体タイムライン（90分・6人）— クラス全員がこの1枚を見る

時刻 長さ 何を 主役
0:00–0:05 5分 セットアップ（3.の①②） 進行役
0:05–0:08 3分 各自が狙う段を記入（3.の③） 全員
0:08–0:53 45分 提案発表＆攻防：6人 × 7.5分 交代
0:53–1:00 7分 投資カードを黙って配分（フォーム入力） 委員
1:00–1:10 10分 開票・リーダーボード確定・Go/見送り発表 進行役
1:10–1:20 10分 総括＋レッドチーム点の確定＋アーカイブ保存 進行役

進行役はスマホのタイマーで各ブロックを区切るだけ。中身の判断はしません。

5. 1人分7.5分の回し方（全員が覚える共通の型）

発表者が1人ずつ、この順番で回します。進行役は声かけとタイマーだけ。

0:00–3:00  提案発表（本番は1枚だけを画面共有）
            └ 委員は「聞きながら」反論カードを書き始める
3:00–4:00  委員が挙手 → 進行役が2人を指名して質問（反論）
4:00–6:00  発表者が防御（ここで初めてバックアップ1枚を出してよい／組み替えてよい＝L3行動）
6:00–6:30  「この反論は成立？」→ 委員の挙手投票（過半数で成立）
6:30–7:30  委員が投資カードの“仮置き”とルーブリック採点をメモ

プレゼンの枚数ルール：本番は 1枚固定。最初から2枚目以降は出さない。
バックアップは1枚まで任意で、委員の反論に答えるとき"だけ"出せる（典型：バイアス検証グラフ）。
詳しくは → [[phase3_役割マニュアル_発表者]]「枚数のルール」。

この回で必ず起きること（1つも飛ばさない）

提案発表 → 反論 → 防御 → 成立投票 → （仮）投資配分

どれか1つでも抜けると、あとで点が付けられません。進行役は「5つ揃ったか」だけ確認。

6. 自走チェックポイント（迷子防止のゲート）

先生がいないと、流れが崩れがち。各ブロックの最後に、進行役が次の1問を声に出して確認してから進みます。

ゲート 進める前に確認する1問 NGなら
各提案発表の終わり 「反論カードは最低1枚出た？ 成立投票はした？」 出てないなら「定番反論」を促す（→ 投資委員マニュアル）
全提案発表の終わり 「全員ぶんの1枚は保存した？」 抜けてたら今すぐ画像/PDF化
投資のあと 「全員8枚ぜんぶ使った？ 自分の案に入れてない？」 ルール違反は配り直し
開票のあと 「60万ライン超えはどれ？ リーダーボードに反映した？」 反映してから締めへ

ゲートは“良し悪しの判定”ではなく“抜け漏れチェック”。だから進行役にもできます。

7. 役割マニュアルへ（自分の役の詳しい動き）

全体像をつかんだら、自分の役のマニュアルを開いてください。動き方はそこに全部あります。

あなたの役 一言 開くもの
進行役 時間と段取りを回す（判定しない） [[phase3_役割マニュアル_進行役]]
発表者（事業部） 1枚で提案し、投資カードを勝ち取る [[phase3_役割マニュアル_発表者]]

進行役以外は、発表者と投資委員を1人で兼ねます。自分の番は発表者、それ以外は委員。

8. 先生がいなくても判定できる仕組み（権威レス早見表）

「これ、先生がいないと決められないのでは？」と詰まりそうな場面は、全部この置き換えで解決します。

決めたいこと 先生がいなくても、こう決める
どの反論を質問するか 委員の 挙手で2件 選ぶ
その反論は成立か（+3） 委員の過半数票で成立
発表者は組み替えたか（+5） 観察事実（提案を変えた／変えてない、だけ）
案の順位・調達額 フォームの自動集計（人の判断ゼロ）
Go か 見送りか 60万円以上か未満か、それだけ
時間管理 進行役＋スマホタイマー

ポイント：「えらい人が決める判定」を1つも残さない設計です。だから自走できます。

9. 詰まったときのQ&A（自走トラブル対応）

困った場面 こうする
進行役を誰もやりたがらない じゃんけん。その回いちばん発表に自信がない人が司会に回ると公平
反論が誰からも出ない 投資委員マニュアルの「定番反論チェック」を1つ読み上げて促す
成立投票が同数で割れた 同数は不成立（0点）。過半数＝賛成が半分より多い、で固定
時間が押した 防御を優先し、次の人の提案発表を少し短くして取り戻す
4人/7人/8人で人数が違う 下の「10. 人数別の調整」で1人あたりの尺を変える
言い合いがヒートアップ 進行役が「データに戻ろう」と一声。人ではなく分析を攻めるのがルール
集まれない／オンライン 「11. 非同期フォールバック」へ

6人 7.5分 標準

ミニ委員会（コマ15-17 / 18-21）＝練習版

  • 本番（コマ25-28）の前に、3.5分/人（提案発表2分＋反論1件＋防御1分）で軽く回す練習回。
  • 自己採点は省略。攻撃の作法とバイアス反論のリハーサルが目的です。

11. 終わりに必ず残すもの（＝先生への引き継ぎ・アーカイブ）

先生はライブにいません。 だから先生が成績（ルーブリック点＝Phase3の50%）を付けられるのは、
君たちが残したアーカイブだけです。進行役が中心になって、1フォルダにまとめてください。

  • 各提案発表の 本番1枚（PDFか画像）＋出したならバックアップ1枚も
  • 投資カード集計（案ごとの調達額・Go/見送り）
  • 反論カード（全部）
  • 可能なら 録画

これが無いと点が付きません。ここだけは絶対に死守。終了前のゲート（6.）で必ず確認を。

非同期フォールバック（集まれないとき）

  • 録画した提案発表＋予算配分フォーム＋コメントで1人1反論に置き換える。
  • タイムボックスは消えても、「希少な予算・順位・反論必須」の力学はそのまま再現できます。

12. 一言まとめ

先生のガイダンスが終わったら、この1枚が君たちの司令塔。
進行役を決める → タイムラインを流す → 判定は多数決と事実 → アーカイブを残す。
順位を競うより、自分の段（L1→L2→L3）を1段でも上げることを狙え。
