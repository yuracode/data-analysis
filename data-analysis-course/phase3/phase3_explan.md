## 1. この講座でいう「問い」とは

ふだんの会話の「質問」とは少し違う。**分析して答えを出すと、次の行動が決まる一文**のこと。

> 「問い」＝ **データで答えられて、答えが出たら誰かが動ける質問**

### 要望・疑問との違い（FitLoopの例）

| | 例 | なぜダメ／良い |
|---|---|---|
| ❌ ただの要望 | 「継続率を上げたい」 | 願望であって質問じゃない。何を調べればいいか決まらない |
| ❌ ザルな問い | 「どうすれば継続率が上がる？」 | 漠然としすぎて、どんな答えでも当てはまる＝動けない |
| ✅ 良い問い | **「プッシュ通知をONにした層は、OFF層より継続率が高いか？」** | YES/NOがデータで出る。出たら「通知を勧める施策を打つか」が決まる |

### 良い問いの3条件
1. **データで答えられる**（手元のカラムで検証できる）
2. **YES/NO や どっち、がはっきり出る**（あいまいに終わらない）
3. **答えが出たら行動が変わる**（＝意思決定に効く）

### なぜ大事か：問いが決まると調べることが自動で決まる
```
問い：「あと一押し層は、どの変数で継続層と一番くっきり分かれるか？」
   ↓ これが決まると…
何を見るか    → 各変数を層別して比較する
どう確かめるか → 差を検定する（偶然じゃないか）
何を見せるか   → 一番差が出た1枚のグラフ
```
逆に問いが曖昧だと、データをいじり回しても「で、結局なんなの？」で終わる。

### 問いは大きいものを小さく割る
```
経営層の要望（問いですらない）
「継続率を改善したい」
        ↓ 翻訳
主問い：3か月継続率に最も影響する要因は何か？
        ├ サブ問い：1か月目の運動頻度は継続に影響するか？
        └ サブ問い：通知機能を使う人は継続率が高いか？
```
**要望（願望）→ 主問い → サブ問い** と降りていき、一番下が「データで一発で答えが出るサイズ」になったら、それが分析の単位になる。

## 2. 手法は「学ぶ」のではなく「問いのために取りに行く」

### どういう意味か
ふつうの勉強は「t検定を習う → 使える場面を探す」という順番（**手法が先**）。
この講座は逆で、「**この問いに答えたい → だからこの手法を取りに行く**」（**問いが先**）。

> 手法は教科書という**道具箱**に並んでいる。
> 全部を順番に覚えるのではなく、**目の前の問いに必要な道具だけを取り出して使う**。

### なぜこの順番なのか
- 受講生はもう sklearn なども触れる経験者。**手の動かし方**はすでに持っている。
- 足りないのは「**どの道具を選ぶか**」と「**出た結果をどう読むか**」の判断。だからそこだけ鍛える。
- 手法から入ると「習った手法を無理やり使う」ことが目的化し、問いとズレる。問いから入れば、手法は必然になる。

### 「取りに行く」具体イメージ（FitLoop）

| 目の前の問い | 取りに行く手法 | なぜそれか |
|---|---|---|
| あと一押し層はどんな人か？ | RFM／ピボット集計／k-means | 「層に分ける」のが目的だから |
| ON層とOFF層で継続率は違う？ | 二群比率の検定・カイ二乗 | 「差があるか」を白黒つけたいから |
| その差は偶然じゃない？ | p値・有意差検定 | 「偶然で説明できないか」を確かめたいから |
| 継続に効く要因はどれ？ | 分類モデル＋SHAP・重要度 | 「どの変数が効くか」を説明したいから |
| この施策は予算内で割に合う？ | ROI試算 | 「Goかどうか」を金額で言いたいから |

ポイントは、表の**右の列が「問いの言い換え」になっている**こと。手法は「問いを別の言葉で言ったもの」にすぎない。

### 手法を選ぶときの自問
1. **この問いは何を知りたい？**（差を見たい／分けたい／予測したい／関係を見たい）
2. **その目的にいちばん合う道具はどれ？**（複数の候補を比べる）
3. **なぜ他の道具を選ばなかった？**（データ・粒度・解釈しやすさのどれが決め手か）
4. **出た結果は、最初の問いに答えているか？**（手法のための分析になっていないか）

> ねらい：手順書を渡さないのは、この「**選ぶ・解釈する**」を自分でやらせるため。
> 実装はできる経験者だからこそ、足りない「選択と解釈の経験」だけを鍛える。

## 4. 一言まとめ

> **問い**は「データで答えが出て、出たら動ける一文」。願望を研ぎ澄ましたもの。
> **手法**はその問いに答えるために道具箱から取りに行くもので、覚える対象ではない。
> 「問いを立てる・分解する」は Ch3・Ch4、「どの手法を取りに行くか」は Ch4.4／Ch8.13 が効く。
