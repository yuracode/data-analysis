# 【役割マニュアル】進行役（ファシリテーター）

> あなたは今回の投資委員会の **進行役** に割り当てられました。
> この授業に講師はいません。**時間と段取りを回すのがあなたの仕事**です。
> 詳しいルールの出典：[[phase3_投資委員会_進行台本]]

---

## あなたの役割（一言）

台本どおりに**時間を進めるだけ**。良し悪しの判定はしません。
判定は「委員の多数決」と「観察事実」が決めます。あなたは**審判ではなく司会**です。

> その回、あなたは提案発表しません。だから中立に進行できます。

---

## ゴール（これができれば成功）

- 90分を時間どおりに回しきる。
- すべての提案発表で「提案発表 → 反論 → 防御 → 成立投票 → 投資カード配分」が**1つも抜けない**。
- 最後に**アーカイブ**（1枚・集計・反論カード・録画）を残す。← 講師の採点材料はこれだけ。

---

## 持ち物

- [ ] スマホのタイマー
- [ ] 投資カード（1枚5万円・8枚×人数分）または配分フォームのリンク
- [ ] リーダーボード用スプレッドシート（投影できる状態）
- [ ] 反論カードの束、または共有スプレッドシート
- [ ] 自己採点シート（人数分）

---

## タイムライン（90分・6人の場合）

### ① 開始前（0:00–0:08）
- [ ] 投資カードを各委員に**8枚ずつ**配る（フォームならリンク共有）
- [ ] リーダーボードを開いて投影
- [ ] 自己採点シートを配り、「**今日狙う段（L1/L2/L3）を書いてください**」と伝える（3分）

### ② 提案発表＆攻防（0:08–0:53／1人7.5分 × 6）
各提案発表で、このセリフどおりに進めてください 👇

```
「○○さん、提案発表どうぞ。3分です」              ← タイマー3分スタート
（3分経過）「はい、ここまで」
「委員のみなさん、反論カードを出してください」
「質問したい人、挙手を。…この2人にお願いします」   ← 挙手で2件選ぶ（1分）
「○○さん、防御どうぞ。2分です」                ← タイマー2分
（2分経過）
「この反論は“成立”だと思う人、挙手を」           ← 過半数なら +3 を記録
「○○さんは提案を組み替えましたか？」            ← 変えた＝観察事実で +5 を記録
「では次、△△さんどうぞ」
```

> 迷ったら：**過半数 = 賛成が半分より多い**。同数なら「不成立（0点）」。

### ③ 投資（0:53–1:00）
- [ ] 「投資カードを配分してください。7分です」
- [ ] フォームを締める

### ④ 開票（1:00–1:10）
- [ ] 各案の調達額を集計（自動集計）
- [ ] **60万円以上 = Go、未満 = 見送り**をリーダーボードに表示
- [ ] レッドチーム点（+3／+5）を合計して表示

### ⑤ 締め（1:10–1:20）
- [ ] 1位を確認、ひとことコメントを募る
- [ ] **アーカイブを保存**（下記）
- [ ] 次回の宿題を伝える：「自己採点と講師採点の差分を埋めてくること」

---

## アーカイブ（絶対に残す・講師の採点材料はこれだけ）

1フォルダに入れて保存：
- [ ] 各提案発表の**1枚**（PDFか画像）
- [ ] **投資カード集計**（調達額・Go/見送り）
- [ ] **反論カード**（全部）
- [ ] 可能なら**録画**

> これが無いと講師がルーブリック点（成績の50%）を付けられません。ここだけは死守。

---

## やってはいけないこと（NG）

- ❌ あなたが「良い／悪い」を判定する → 判定は多数決と観察事実
- ❌ 時間を延ばす → 押したら**防御を優先し、提案発表を短く**して取り戻す
- ❌ アーカイブを残し忘れる

---

## 困ったら

| 状況 | どうする |
|---|---|
| 票が割れた | 過半数=賛成多数。同数は不成立(0) |
| 時間が押した | 防御を優先、提案発表の尺を縮める |
| 反論が出ない | 「F.反論カードの定番チェック」を読み上げて促す |
| 人数が4/7/8人 | 台本B「人数別の調整」表のとおり1人あたり尺を変える |
