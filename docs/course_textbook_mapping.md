## 既知のギャップと方針（ハイブリッド方針で対応中）

1. **Ch4の核心が体系的に未カバー** … 縦/横の論理、2つのSo What?、帰納・演繹、MECE、問題解決ステップ（WHERE→WHY→HOW）。
   コマ7でロジックツリー・So What?・バイアスは触れているが、思考フレーム全体は未整理。
   → **思考法コマを新設予定**（[`phase2/07b_thinking_methods.md`](../data-analysis-course/phase2/07b_thinking_methods.md) 素案）。
2. **章ラベルのズレ** … コマ8（旧Ch4表記→実はCh5）、コマ10（旧Ch5表記→実は発展）を修正済。各コマ冒頭に「教科書との対応」注記を追加した。
3. **Ch5マーケ基礎の置換** … 教科書のSTP・消費者理解・行動経済学は、本講座ではファネル/RFM/コホートに再構成。コマ8の注記で各自参照を案内。
4. **Ch8多変量解析** … 因子分析・クラスター・コンジョイント・因果推論（教科書8.5-8.14）の扱いをコマ32で要確認。UdemyのML（分類/回帰）とは別系統。
